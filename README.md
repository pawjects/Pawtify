<h2 align="center">✨ About The Project</h2>

<p align="center" style="max-width: 800px; margin: auto;">
  Built for those who want a distraction-free audio experience, <b>Pawtify</b> is a lightweight, browser-based music player. Inspired by modern, dark-themed aesthetics and sleek UI principles, it runs entirely statically. Whether you're dropping in a high-energy remix or reading along to your favorite song lyrics, the interface stays out of your way, delivering blazing fast load times with zero backend clutter.
</p>

<br>

<p align="center">
  <b><a href="https://pawtify-meow.netlify.app/">🌐 Try the Live Demo Here!</a></b>
</p>

<h2 align="center">🎨 Core Features & Capabilities</h2>

<!-- Detailed Feature Grid using HTML Tables for perfect alignment -->
<table align="center" width="100%" border="0" cellpadding="10" cellspacing="0">
  <tr>
    <td width="50%" align="center">
      <h3>🎵 Seamless Playback</h3>
      <p>Full audio control with smooth play/pause toggling, track skipping, and a precision progress bar for seeking through tracks.</p>
    </td>
    <td width="50%" align="center">
      <h3>📜 Integrated Lyrics</h3>
      <p>Read along to your favorite tracks. The UI allocates dedicated space for lyric viewing without cluttering the main controls.</p>
    </td>
  </tr>
  <tr>
    <td width="50%" align="center">
      <h3>🖼️ Dynamic Album Art</h3>
      <p>Beautiful, rounded-corner thumbnail displays that update instantly to reflect the currently playing song's artwork.</p>
    </td>
    <td width="50%" align="center">
      <h3>🔊 Audio Control</h3>
      <p>Integrated volume sliders and mute toggles to ensure your listening experience is perfectly balanced right from the browser.</p>
    </td>
  </tr>
  <tr>
    <td width="50%" align="center">
      <h3>⚡ Fully Static & Zero Backend</h3>
      <p>Powered entirely by core web technologies (HTML, CSS, JS). No databases, no login screens, no APIs required. Just pure performance.</p>
    </td>
    <td width="50%" align="center">
      <h3>📱 Mobile-First Responsiveness</h3>
      <p>Fluidly adapts to any screen size. Buttons, progress bars, and lyrics perfectly format for desktop monitors and mobile devices alike.</p>
    </td>
  </tr>
</table>
