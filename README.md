✨ Overview

Pawtify is a next-generation frontend music streaming platform designed to deliver a seamless and ad-free listening experience.

Inspired by modern music platforms while maintaining its own identity, Pawtify focuses on speed, simplicity, and accessibility. The platform provides a clean interface where users can discover music, search tracks, manage their library, and enjoy uninterrupted streaming.
